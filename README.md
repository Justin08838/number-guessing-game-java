# Number Guessing Game (Java)

## Description
A **console-based number guessing game** written in Java.  
The program generates a random number between **1 and 100**, and the user has to guess it.  
It provides hints like "Too high" or "Too low" until the correct number is guessed.

This project demonstrates **loops, conditional logic, random number generation, and input handling** â€” perfect for beginners learning Java.

---

## Features
- Generates a random number between 1 and 100  
- Provides hints: Too high / Too low  
- Counts the number of attempts  
- Accepts input from the user via console  
- Displays a congratulatory message when the number is guessed correctly  

---

## Technologies Used
- Java  
- Scanner class for user input  
- Random class for random number generation
